# Chuzzle — Jeu de puzzle Android

Clone du jeu Chuzzle sur Android : grille 6×6 avec glissement de lignes/colonnes  
et détection de séries pour marquer des points.  
**Contenu :** partie normale · partie à graine fixe · hard mode · système de score · verrous  
**Stack :** Java · Android

> Projet académique — BUT Informatique
